## 🌟 Overview

This project is a full-stack Uber clone that allows users to **book rides** and captains to **accept and manage rides in real time**.  
It focuses on **real-time ride flow, authentication, and location-based booking experience**.

---

## ⚡ Features

- 👤 **User Authentication**
  - Secure Sign up / Login system

- 🚖 **Captain Module**
  - Separate login for captains
  - Ride acceptance & management

- 📍 **Ride Booking**
  - Pickup & destination selection
  - Fare estimation before booking

- 🔄 **Real-Time Updates**
  - Live ride updates using **Socket.io**
  - Instant communication between user & captain

- 🗺️ **Google Maps Integration**
  - Location-based services
  - Pickup & destination handling

- 📱 **Responsive UI**
  - Optimized for mobile devices
  - Built with **React + Tailwind CSS**

- 🛡️ **Protected Routes**
  - Role-based authentication system
